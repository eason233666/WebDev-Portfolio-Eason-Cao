# WebDev-Portfolio-Eason-Cao

My Web Development Portfolio

This repository contains my personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections on scroll
- Project showcase with interactive cards
- Contact form with validation
- Modern and clean UI

## Structure

- `index.html`: Home page with hero section and previews
- `about.html`: About page with skills and background
- `projects.html`: Projects showcase page
- `contact.html`: Contact page with form and information
- `styles/style.css`: Main stylesheet with responsive design
- `scripts/script.js`: JavaScript for interactivity and animations
- `images/`: Image assets (add your profile photo here)
- `README.md`: This file

## Customization

1. Replace placeholder content with your information
2. Add your profile image to the `images/` folder
3. Update contact information in `contact.html`
4. Modify colors and styles in `styles/style.css`
5. Add more projects to `projects.html`

## How to View

Open `index.html` in your web browser to view the portfolio.
